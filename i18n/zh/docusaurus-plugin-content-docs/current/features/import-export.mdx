# 完全掌控：导出和导入您的数据
<NLevelAnalytics />

在 positive-intentions，我们理解您的数据属于您。我们重视您控制、移动和管理个人信息的权利。这就是为什么我们在我们的应用程序中构建了一个简单的导出和导入功能，让您完全掌控您的数据。

只需几个简单的点击，positive-intentions 允许您随时导出所有数据。这意味着您可以随时备份所有聊天、视频通话历史和个人资料信息。导出文件是通用可读的，透明的，确保您清楚了解您正在管理的数据。

同样，我们的导入功能允许您快速轻松地恢复数据。要换设备？没问题。只需导入先前导出的数据，您将立即恢复您的整个 positive-intentions 历史、个人资料和偏好设置。

导出和导入数据的能力不仅是一个功能，更是我们对隐私和透明承诺的体现。这意味着您的信息不会被锁定在专有系统中，而是完全按照您的需求进行访问和移动。

因此，在聊天、通话和连接时，您可以放心，知道您对数据拥有完全的自主权。这就是 positive-intentions 提供的自由和掌控力。
