## 创建个人资料

开始使用非常简单。按照以下步骤设置您的个人资料：

1. **自动生成用户 ID：** 您的应用程序会为您生成一个独特且密码学安全的 ID，用于连接和识别您自己。这确保了安全和私密的体验。

2. **阅读并接受条款：** 在您使用连接功能之前，我们要求您阅读并同意我们的条款和条件。您的安全和隐私对我们非常重要。

3. **选择用户名：** 设置一个代表您的用户名。这是您的朋友在聊天中认出您的方式。

4. **添加个人头像：** 通过添加个人头像让您的个人资料更加生动。点击您的默认图像并上传反映您个性的图片。

5. **应用程序偏好设置：** 自定义您的应用程序偏好设置，包括通知设置，以与您的沟通风格匹配。

## 个人资料设置

通过以下可自定义的设置来个性化您的聊天体验：

- **用户名：** 您可以随时更改用户名，使事物保持新鲜。

- **个人头像：** 随时更新您的个人头像。

- **连接 ID：** 在个人资料设置中自定义您的连接 ID。此 ID 帮助您在保持隐私的同时与他人建立连接。

- **应用程序偏好设置：** 调整通知设置和其他应用程序偏好，以适应您的喜好。

## 安全提醒

- **端到端加密：** 您的消息通过端到端加密进行保护，但请记住安全性还取决于您连接的人。只与您信任的人进行聊天。

- **节点信任：** 对于您不认识的节点，要保持谨慎。坚持与可信任的联系人保持连接，以确保安全的聊天环境。

- **网络安全：** 请记住，即使有强大的加密，也最好避免在互联网上分享敏感信息。
