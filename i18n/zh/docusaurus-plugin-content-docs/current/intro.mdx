# 欢迎
<NLevelAnalytics />

欢迎来到 positive-intentions，数字聊天领域的新成员。我们的应用致力于提供安全、私密的聊天体验，您可以轻松地通过您的网络浏览器访问。

如果您喜欢安全、基于网络的聊天应用的想法，那么您来对地方了。

与许多聊天应用不同，我们不依赖于庞大而复杂的后端服务器。我们使用现代浏览器技术和 peerjs-server 来使我们的应用运行。这意味着当您与某人聊天时，这是您和他们之间的直接联系 - 没有其他人插足其中。

positive-intentions 的真正亮点在于它生存在网络上。是的，会有一些权衡，但我们认为这是值得的。通过我们的应用，您可以立即开始聊天，无论您使用什么浏览器或操作系统。而且无需放弃任何酷炫的功能。

positive-intentions 背后的魔法是我们独特的点对点（P2P）架构。我们自己构建了这个架构，为我们的应用提供动力，使所有这些炫酷的功能成为可能。

我们倾注了很多心血来构建 positive-intentions。我们希望它能够帮助您和许多其他人与朋友、家人以及任何您想联系的人安全聊天。

值得注意的是，positive-intentions 还在不断发展中。我们仍在开发一些功能，未来还会有更多令人兴奋的变化。敬请期待！

准备好开始了吗？我们很乐意听取您的意见。您想在 positive-intentions 中看到哪些功能？让我们一起把它打造得更好。您可以通过我们的 [Reddit](https://www.reddit.com/r/positive_intentions//) 联系我们了解有关我们应用的更多信息。
